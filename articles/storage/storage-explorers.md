# <a name="azure-storage-client-tools"></a>Outils clients stockage Azure

Les utilisateurs de stockage Azure comptez puissent afficher/interagir avec leurs données à l’aide d’un outil client stockage Azure. Dans les tableaux ci-dessous, nous indiquons une série d’outils qui vous permettent de faire cela. Nous placez un « X » dans chaque bloc si elle offre la possibilité soit énumérer les et/ou accéder à l’abstraction de données. Le tableau indique également si les outils est disponible ou non. « D’évaluation » indique qu’il existe une version d’évaluation gratuite, mais complète du produit n’est pas disponible. « Y/N » indique qu’une version est disponible gratuitement, alors qu’une autre version est disponible à l’achat.

Nous avons fourni uniquement un instantané des outils clients stockage Azure disponibles. Ces outils peuvent continuent d’évoluer et fonctionnalités. Si des corrections ou des mises à jour, vous devez laisser un commentaire pour nous faire connaître. La même a la valeur true que si vous connaissez des outils doivent être ici - nous serait satisfaits les ajouter.
